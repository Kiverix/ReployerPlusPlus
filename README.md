# Reployer++

The next generation of Reployer is now here!
In this new and improved version, you are no longer bound to only monitoring cge7-193, as long as you have a valid IP that is.

You can set profiles of servers you want to monitor, with corresponding games, optional FastDL download urls, and the ability to connect to the server by the press of a button (if you have the required game installed that is)

Stay tuned for the first release!
